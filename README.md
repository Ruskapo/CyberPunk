# CyberPunk

Лендинг в стиле киберпанк: семантическая вёрстка, адаптив, CSS-эффекты и анимации.

## Демо
https://ruskapo.github.io/CyberPunk/

## Технологии
- HTML5 (semantic)
- CSS3 (Flexbox / адаптив)
- CSS animations / transitions

## Функционал
- Адаптивная сетка для мобильных/планшетов/десктопа  
- Эффекты наведения и плавные анимации  
- Аккуратная типографика и контрастные цвета

## Как запустить локально
Открой `index.html` в браузере  
или подними простой сервер:

# вариант через VS Code: Live Server
# либо через Node (поставит локальный сервер в текущей папке):
npx serve .
